# CARICKAT 1.0
## Carnet de croquis vivant numérique

---

# Vision du projet

Carickat 1.0 est un site pensé comme une fusion entre :

- une galerie d’artiste moderne,
- un atelier de dessin,
- un carnet de croquis vivant.

Le but n’est pas de créer un simple portfolio froid.
Le site doit donner l’impression d’entrer directement dans l’esprit et l’atelier du dessinateur.

Le visiteur doit ressentir :
- les recherches,
- les essais,
- les annotations,
- les traits,
- les hésitations,
- les déformations,
- les œuvres terminées,
- et le processus derrière chaque dessin.

---

# Direction artistique

## Ambiance générale

Le site repose sur une esthétique :

- papier blanc cassé,
- texture légère,
- carnet de croquis,
- atelier vivant,
- galerie artistique moderne,
- annotations manuscrites,
- œuvres légèrement inclinées,
- éléments dessinés à la main.

Le design doit rester :

- lisible,
- respirant,
- fluide,
- mobile friendly,
- élégant malgré l’aspect organique.

# Style visuel

## Palette

### Papier
- blanc cassé,
- crème,
- beige clair.

### Encre
- noir doux,
- brun,
- rouge sombre.

Le site doit éviter :
- néons,
- esthétique corporate,
- ambiance futuriste,
- surcharge visuelle.

---

# Typographies

## Texte principal
- Inter
- Manrope

## Style carnet
- Caveat

Utilisée uniquement pour :
- annotations,
- notes,
- signatures,
- détails manuscrits.

---

# Éléments visuels importants

Le site doit intégrer subtilement :
- œuvres inclinées,
- cadres irréguliers,
- ombres douces,
- texture papier,
- annotations manuscrites,
- marges vivantes,
- ambiance atelier.

Mais toujours garder :
- lisibilité,
- respiration,
- fluidité.

---

# Responsive

Le site doit être parfaitement lisible sur mobile.

Sur mobile :
- les rotations sont réduites,
- les colonnes deviennent verticales,
- la navigation reste simple.

# Philosophie générale

- une boutique froide,

Carickat 1.0 =

20% atelier artistique
