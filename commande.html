processus clair

Comment se déroule une commande ?

Le but est de garder un processus simple : tu expliques, on clarifie, puis le dessin prend forme dans une direction propre.

01

Tu envoies la base

Une photo, une idée, une personne, une émotion, une référence ou une intention.

Point de départ
02

On valide la direction

On définit le style : drôle, élégant, étrange, brut, détaillé, cadeau ou pièce plus artistique.

Direction claire
03

Je crée le dessin

Le dessin est réalisé avec l’énergie Carickat : trait vivant, déformation, présence et personnalité.

Création finale

valeur

Pourquoi commander une pièce personnalisée ?

Parce qu’un dessin personnalisé peut devenir plus qu’une image : un souvenir, un objet, une blague privée, un symbole, un cadeau ou une trace graphique d’une personne.

L’intérêt de Carickat est de garder le côté humain du dessin : les imperfections, le trait, l’intention, la déformation, et cette impression que l’image a été pensée pour quelqu’un.
