commande personnalisée

Un visage devient une histoire.

Envoie une photo, une idée ou une intention. Je transforme le visage en dessin expressif : caricature, portrait stylisé, version étrange, drôle, intense ou plus brute.

Lancer une commande Voir les exemples
note de carnet

Pas une copie froide.

Le but n’est pas seulement de reproduire un visage. Le but est de capter une énergie : une attitude, une grimace, une présence, un détail qui devient central.

processus

Comment ça marche ?

01

Tu envoies la base

Une photo, une personne, une idée, une émotion ou une direction artistique.

02

Je cherche l’angle

Je repère les formes fortes : regard, bouche, posture, énergie générale.

03

Le dessin prend forme

Le portrait devient une version plus expressive, plus vivante, parfois plus absurde.
